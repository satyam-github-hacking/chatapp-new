# Advanced Real-Time Chat Application

## Working Features
- JWT Authentication Structure
- Real-time WebSocket Chat
- One-to-One Chat
- Group Chat
- File Upload Support
- Online Status Structure
- Typing Indicator Structure
- Seen Status Structure
- Responsive UI
- Dark Mode UI

## Tech Stack
Backend:
- Django
- Django REST Framework
- Django Channels

Frontend:
- React
- Tailwind CSS

Realtime:
- WebSockets

## Backend Run

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Frontend Run

```bash
npm install
npm run dev
```
